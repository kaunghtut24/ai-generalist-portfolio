AI Generalist Portfolio — Starter Kit
=====================================

What you get
- index.html — the site HTML (modern, accessible)
- style.css — responsive stylesheet (mobile-first)
- data.js — simple JSON-like data object to populate projects & certs
- script.js — tiny renderer + mobile nav behavior
- README.md — this file

How to use
1. Unzip the package.
2. Replace placeholder text in index.html (your name, links).
3. Update data.js with your real projects and certification links.
4. Optionally host on GitHub Pages or any static hosting.

Contact form
- index.html uses Formspree as a simple no-backend form. Replace the 'yourFormId' in the form action with your Formspree form id.

Accessibility & SEO improvements
- Skip link, ARIA labels, semantic elements, focus styles, and Open Graph meta tags are included.
